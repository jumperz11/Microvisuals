| Typography | Inter (headings), monospace (body/quotes) |

### Quote Examples

- "Motivation fades. Discipline persists."
- "Time is a finite resource, but focus is a renewable one."

### Visual Metaphor Patterns

1. **Single object** - One clear focal point
2. **Failure/success state** - Shows consequence visually
3. **No secondary objects** - Metaphor must stand alone
4. **Cold, observational tone** - No advice, just facts

### Quote Layout Formula

```
[Top text]: "What [situation] feels like:"
[Bottom text]: The metaphor punchline (1-2 lines)
```

## Design Principles Learned

1. **Less is more** - Strip away non-essential elements
2. **One idea per visual** - Don't overload with meaning
3. **Object must fail visibly** - The metaphor needs visual proof
4. **Cold tone** - Observational, not preachy
5. **Consistent style** - Same aesthetic across all posts
6. **High contrast** - Black/white for maximum impact

**Profile:** https://www.instagram.com/visualtheory_/
**Followers:** 75K
**Posts:** 416

### Visual Style

| Element | Details |
|---------|---------|
| Graphics | High-quality 3D renders, clean illustrations |
| Color | Black backgrounds, selective color accents |
| Typography | Bottom-positioned, clean sans-serif |
| Layout | Image dominates, text at bottom |

### Key Differences from Neuronvisuals

| Aspect | Neuronvisuals | Visual Theory |
|--------|---------------|---------------|
| Graphics | Flat 2D, Bauhaus | 3D renders, more detailed |
| Text position | Bottom, minimal | Bottom, slightly larger |
| Complexity | Ultra minimal | More visual detail |

### Text Positioning Pattern

```
┌─────────────────────────────────┐
│                                 │
│                                 │
│        [MAIN VISUAL]            │
│                                 │
│                                 │
├─────────────────────────────────┤
│  "Quote line 1"                 │  ← Bottom section
│  "Quote line 2"                 │  ← Clean, readable
└─────────────────────────────────┘
```

### Learnings

- Text goes at **BOTTOM**, not centered
- Adapt text position based on image content
- Leave breathing room between image and text
- Font: Clean sans-serif (Inter works well)

## Image Sizes (2025)

### Best Sizes by Platform

| Platform | Best Size | Ratio | Notes |
|----------|-----------|-------|-------|
| Instagram Feed | 1080 x 1350 | 4:5 | More screen space, better engagement |
| Instagram Square | 1080 x 1080 | 1:1 | Classic, universal |
| Instagram Story | 1080 x 1920 | 9:16 | Full screen vertical |
| X/Twitter | 1600 x 900 | 16:9 | Optimal for timeline |
| X Square | 1080 x 1080 | 1:1 | Works well |

### Universal Size

**1080 x 1080 (1:1)** works on both Instagram and X/Twitter - this is what Neuronvisuals uses.

### App Export Options

| Button | Size | Use For |
|--------|------|---------|
| 1:1 | 1080 x 1080 | Universal (Instagram + X/Twitter) |
| 4:5 | 1080 x 1350 | Instagram feed (more engagement) |
| 9:16 | 1080 x 1920 | Stories (Instagram/TikTok) |

**Note:** For X/Twitter, use 1:1 square - same as Instagram. Top accounts post identical images on both.
